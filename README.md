# sistema-caixa-java
# 🏪 Sistema de Caixa em Java

Sistema de caixa (PDV) simples desenvolvido em Java com interação via terminal.

## 🚀 Funcionalidades

* Cadastro e listagem de produtos
* Carrinho de compras
* Remoção de itens
* Cálculo do total e troco

## 🧠 Tecnologias

* Java
* Programação Orientada a Objetos (POO)

---

Projeto para prática de lógica e estrutura de sistemas.
